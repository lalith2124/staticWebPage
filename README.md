# Carint – Transport Services Website

A static frontend website for a transport and logistics company.

## Pages & Sections

- **Hero** – Banner with tagline and call-to-action buttons
- **Stats** – Key company numbers (shipments, on-time rate, countries, experience)
- **Services** – Air, Truck, Cargo, and Train transport cards
- **About** – Company overview with feature highlights
- **Track Shipment** – Tracking number input form
- **Contact** – Contact form + embedded Google Map
- **Footer** – Links, address, social icons, newsletter subscribe

## File Structure

```
project/
├── index.html       # Main HTML file
├── carint.css       # All styles
└── about-img.jpg    # About section image (add your own)
```

## How to Run

**Option 1 – Terminal**
```bash
# Windows
start index.html

# Mac
open index.html

# Linux
xdg-open index.html
```

**Option 2 – VS Code Live Server**
1. Install the **Live Server** extension in VS Code
2. Right-click `index.html` → **Open with Live Server**

## Built With
